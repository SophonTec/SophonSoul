# SophonSystem
The computational core of Sophon cognitive AI systems.

### Installation
Pre-requirement:
1. python 3.8
2. torch>=1.6.0

Download the source from github:
```bash
git clone https://github.com/SophonTec/SophonSystem
```
Then build the code
```bash
cd SophonSystem && pip3 install -e .
```
